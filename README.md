# Market Commodity Price Bot

An n8n automation that pulls live commodity prices from the market and sends daily updates to Telegram.

### What it does
This workflow automatically:
1. Pulls live market data for Corn, Rice, Soybeans, and Wheat from public APIs
2. Cleans and merges the data
3. Formats it into a readable report
4. Sends it to Telegram on a schedule

### Why
Stop manually checking commodity market prices. Let the bot do it for you.

### Tech Stack
- n8n for automation
- Market Data APIs
- Telegram Bot API

### Setup
1. Import workflow.json into n8n
2. Add your Telegram credentials
3. Activate and let it pull prices daily
